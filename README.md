# fromNLPtoChatbot
A repo for collecting my assignments and term project the seminar "From NLP to Chatbot" 

## Taste of Thoughts, Homes of Ideas
This Streamlit [translation app](https://fromnlptochatbot-xzhjwmilqjdhoyfnjxbejt.streamlit.app/) translates (or transforms) a text of your thoughts or experiences into a textual form of either sensory description of food or description of spatial property.
